# 补充资料提取器（文献调研 + 医院LIS数据分析）

## 职责

读取 `补充资料/` 下的两类高价值补充资料：
- **文献调研** → 充实第三章研发背景
- **医院LIS数据分析** → 本地真实世界证据，是样本量可行性论证的核心

---

## 一、文献调研提取

读取 `补充资料/` 下所有 .md 文件。

### 提取内容（5 类信息）

| # | 类别 | 提取内容 | 用途 |
|---|------|---------|------|
| L-01 | 被测物生物学 | 基因/蛋白结构、亚型、功能、信号通路 | 第三章研发背景 |
| L-02 | 疾病关联 | 被测物与目标疾病的机制关系、预后价值 | 第三章、预期用途论证 |
| L-03 | 诊断价值 | 灵敏度/特异度、cutoff 研究、与其他标志物联检 | 第五章评价指标、第六章可接受标准 |
| L-04 | 流行病学 | 国内外发病率/患病率、高危人群 | 第三章流行病学背景 |
| L-05 | 研究进展 | 最新国内外指南推荐、临床共识 | 第三章、参考文献补充 |

### 提取要点

- 每条信息标注文献来源（作者、期刊、年份），便于第十九章引用
- 优先提取近 5 年的文献
- 区分观点性结论和证据性结论

## 二、医院 LIS 数据分析提取

读取 `补充资料/` 下的文件，按内容识别哪些是 LIS 数据分析。

### 必须提取的 5 类信息

| # | 类别 | 提取内容 | 方案用途 |
|---|------|---------|---------|
| S-01 | **检测总量** | 时间跨度内总检测例数、月均检测量 | 第六章样本量可行性论证 |
| S-02 | **阳性率/异常率** | 阳性比例、按病原体/指标的分层阳性率 | PT 取值依据 |
| S-03 | **浓度/Ct值分布** | 覆盖范围、中位数、分位数 | 法规要求可行性 |
| S-04 | **人群分布** | 年龄/性别/科室分层分布 | 分层分析可行性 |
| S-05 | **样本类型分布** | 各样本类型的占比 | 多样本类型分析可行性 |

### LIS 数据内部判断样本量可行性（不写入方案正文）

> LIS 数据仅供内部决策，不得写入方案正文。方案正文只写干净的统计学论证（P0 + PT + n），绝口不提 LIS 数据。

内部判断逻辑：
```
LIS 数据 → 推算预计可获得阳性样本
    → 与统计估算的阳性样本需求对比
    → 充足/临界/不足
    → 若不足，调整 PT 或增加机构数/延长周期
```

同样，LIS 的浓度分布数据用来内部验证法规隐藏要求的可行性：
- "应覆盖线性范围" → 内部确认
- "应包含弱阳性样本不少于 X 例" → 内部确认
