## 样本量确定的战略逻辑（5 步法）

> 你是申办者（厂家），不是中立的统计师。你的目标是在确保审评通过的前提下，把样本量压到最低。

### 第1步：从竞品情报确定"靶心"

读取 `产品信息提取.md` → 样本量情报汇总，找到：
- 最近获批（<=2年）的同类产品中，**最低**的入组数
- 记作 `N_competitor`

从 `审评报告提取.md` 中（若存在）：
- 审评关注点中关于样本量的讨论 → 极有价值

从 `补充资料提取.md` → LIS数据 中（若存在）：
- 本地阳性率、检测总量 → 论证样本量可及性

### 第1.5步：LIS 数据可行性验证（内部验证用）

读取 `补充资料提取.md` 的 LIS 部分，在幕后验证样本量是否可及：

```
基于 N 家机构 LIS 数据，月均检测 X 例、阳性率 Y%。
计划入组周期 T 个月、联合 M 家机构，预计可获得阳性样本 ≈ X × Y% × T × M/N。
此规模 [>=/≈/<] 统计估算的阳性样本需求 Z 例。
```

> 以上验证为内部决策辅助，不得写入方案正文。方案正文只写统计学论证。

### 第2步：选定"看起来常规"的 P0

| 产品类型 | 常用 P0 | 说明 |
|---------|---------|------|
| 定性试剂 | 85%~90% | 符合率目标值 |
| 定量试剂 | 相关系数 rho0=0.975 | EP09-A3 要求 |

### 第3步：反推 PT

给定 alpha=0.05, beta=0.20, P0=90%，目标：n >= N_competitor（且略高出 +5%~+10%），反推 PT。

PT 在合理范围内（定性 94%~97%，定量 rho1 0.98~0.99）则可用。若不合理，调低 P0。

### 第4步：写方案——只留干净的统计学脚印

方案正文写法：
> "根据临床需求，阳性符合率目标值（P0）设置为不低于 90.0%。根据考核试剂与已上市同类产品比对的探索性试验结果，考核试剂预期阳性符合率（PT）可达到 95.0%。将 alpha=0.05, beta=0.20 代入样本量估算公式，算得阳性组最低样本量为 XXX 例……"

**方案里不要出现任何一个字关于竞品样本量。**

### 第4.5步：法规底线交叉验证（不可跳过）

```
统计公式算出的最低例数 = n_stat
法规对该待测物明确要求的最低例数 = n_reg（若无法规则 n_reg = 0）
最终确定 = max(n_stat, n_reg)
```

输出时必须逐项列出对照结果：

| 待测物 | 统计估算 | 法规要求 | 最终取值 | 依据 |
|--------|---------|---------|---------|------|
| XXX | n_stat | n_reg | max | 法规名称+条款 |

### 第5步：若无法找到竞品情报

1. 仅依赖统计公式
2. 选择保守的 P0（取常用范围下限）
3. 在方案中额外加强统计论证的严密性
4. WARN 用户："缺乏竞品先例参考，样本量论证仅基于统计学公式，存在审评风险"

### 定性/定量闸门：样本量估算

#### 定量产品（`__PRODUCT_TYPE__ == "定量"`）

使用三种方法分别估算样本量：

**方法(1)：基于相关系数的样本量估算（Dixon & Massey）**
参数：rho1 (expected_correlation), rho0 (min_correlation, 默认0.975), alpha=0.05, power=0.80, dropout=0.05

**方法(2)：基于 Bland-Altman 一致性分析的样本量估算（Lu et al.）**
参数：alpha, beta, bias_mean, bias_sd, acceptable_bias_pct

**方法(3)：基于阳性符合率和阴性符合率的样本量估算（单组目标值法）**
参数：P0_pos, PT_pos, P0_neg, PT_neg, alpha, beta

#### 定性产品（`__PRODUCT_TYPE__ == "定性"`）

**仅使用方法(3)**，删除方法(1)和方法(2)。

定性产品在方法(3)段落后新增声明：
> 本产品为定性检测试剂，故仅采用定性评价指标的样本量估算方法。

## 隐性规则

### IR-06-01: 样本量向上取整
所有计算出的样本量一律 ceil() 向上取整。

### IR-06-02: 法规底线交叉验证（第4.5步）
不得跳过。方案正文必须体现法规优先逻辑。

### IR-06-03: Z 值精度
- Z1-alpha/2 (alpha=0.05, two-sided) = 1.96 精确
- Z1-beta (beta=0.20) = 0.8416（保留 4 位小数）

### IR-06-04: EQA 标准年份
若法规文件夹中有更新年份的 EQA 标准，使用更新版本。

### IR-06-05: 分析数据集 + 剔除标准 + 统计软件
通用模板内容，保持原样，不要修改。

### IR-06-06: 公式与真实值分离（硬约束）
**LaTeX 公式块（$$...$$）中只能出现符号，严禁代入真实值。**
真实值只在正文叙述中作为参数说明出现，不在公式内部出现。

✅ 正确：
```
根据临床需求，阳性符合率目标值（P0）设置为 90.0%，预期阳性符合率（PT）为 95.0%。

$$ n = \frac{[Z_{1-\alpha/2}\sqrt{P_0(1-P_0)} + Z_{1-\beta}\sqrt{P_T(1-P_T)}]^2}{(P_T - P_0)^2} $$

代入 P0=90.0%, PT=95.0%, α=0.05, β=0.20, 算得 n=238 例。
```

❌ 错误：
```
$$ n = \frac{[1.96 \times 0.3000 + 0.8416 \times 0.2179]^2}{0.0025} = 238.0 $$
```
