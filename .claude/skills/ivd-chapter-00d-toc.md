# IVD Chapter 00d — 目录

## 数据源

读取 `output/<name>/temp/` 下所有章节文件（01_ ~ 19_），提取所有 `#`、`##`、`###` 标题。

## 生成规则

1. 扫描每个文件，提取 `#` 一级标题 → 生成顶层条目
2. 扫描 `##` 二级标题 → 生成缩进子条目
3. 扫描 `###` 三级标题 → 生成更深缩进条目
4. 每条目生成 markdown 锚点链接：`[标题文本](#anchor-id)`
5. 锚点 ID 使用原模板中的 `<a id="_Toc...">` 标签 ID（若存在），否则从标题文本生成
6. 若 `__PRODUCT_TYPE__ == "定性"`，目录中不应出现"离群值检验""相关性分析""偏差图""回归分析""Bland-Altman""医学决定水平"等条目

## 验证

- 目录条目与正文实际标题一一对应（数量、文字一致）
- 所有锚点链接有效
- 无 `【XXX_*】` 残留
